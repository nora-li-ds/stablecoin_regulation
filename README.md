# SECU0057 Stablecoin Regulation Project

## Project Aim

This project examines whether media and crypto-community anxiety around stablecoin regulation can help explain changes in Ethereum stablecoin transfer behaviour, making AML-related risk signals more transparent and interpretable.

本项目分析稳定币监管事件前后，主流媒体和加密社区的焦虑情绪是否能帮助解释以太坊稳定币转账行为的变化，从而让反洗钱相关风险信号更加透明和可解释。
Data sources:
- Guardian Open Platform
- Reddit
- Ethereum stablecoin transfer data

This repository is used as a learning and project log.
